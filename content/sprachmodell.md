# Sprachmodell

Was das Modell mit dem Eingang macht — die eigentliche Sprach-Operation. Das Modell braucht immer einen Input aus dem vorigen Schritt.

## Sprache → Text
Aus einer Sprachaufnahme wird Text — Transkription, Diktat, Mitschrift.

## Text → Sprache
Aus einem Text wird Audio — Vorlesen, z.B. für Barrierefreiheit.

## Bild / Scan → Text
Aus einem Bild oder Scan wird lesbarer Text (OCR).

## Zusammenfassen
Aus viel Text wird wenig — Kernaussagen behalten, Details weglassen.

## Umformulieren
Einen Text in Ton, Länge oder Struktur ändern — kürzer, einfacher, förmlicher, gegliedert.

## Übersetzen
Einen Text in eine andere Sprache übertragen — TR, AR, UK, EN, FR …

## Entwurf schreiben
Aus Stichpunkten oder Notizen wird ein zusammenhängender Fließtext.

## Informationen herausziehen
Aus einem Text Fristen, Namen, Beträge oder Kernaussagen extrahieren.

## Kategorisieren / Zuordnen
Einem Eingang Thema, Dringlichkeit oder Zuständigkeit zuordnen.

## Vergleichen / Prüfen
Einen Text gegen Vorlage, Checkliste oder zweites Dokument abgleichen — was fehlt, was ist anders?

## Fragen beantworten
Konkrete Fragen an ein Dokument stellen und Antwort erhalten.

## Anonymisieren
Aus einem Text Namen, Adressen und persönliche Daten entfernen oder ersetzen.

## Vorlage ausfüllen
Vorgefertigte Felder in einem Template mit Daten aus dem Eingang befüllen.

## Stimmung erkennen
Erkennen, ob ein Text positiv, negativ, neutral oder dringend ist.
